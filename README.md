正如悦读FM所表达的【当好的文字遇上好的声音】，悦读FM提供了一个很好的文章在线收听平台，在这里您可以重拾宁静、思考人生、感悟人生.

我，作为悦读FM的小粉丝，一直苦于悦读FM没有手机客户端，而作为一位手机APP开发者，觉得非常有必要出一款APP，于是该APP就诞生了.
